# CONTEXT_SEED

O **Gestor IA** é um sistema **PHP 8.1+ MVC Customizado** focado em mentoria corporativa via IA. A arquitetura prioriza **Injeção de Dependência** automática via Router, padrões **SOLID**, e segurança robusta com **RBAC (Admin, Manager, Employee)**, **CSRF** e **PSR-12**. O banco de dados **MySQL** gerencia uma hierarquia de gestão e um workflow de relatórios mensais (`draft` -> `submitted` -> `approved`). O frontend utiliza **Tailwind CSS** com estética minimalista (Indigo/Slate). O sistema evolui através de **User Insights** extraídos por IA para personalizar a experiência de mentoria. Todas as interações futuras devem respeitar essa estrutura MVC sem frameworks, manter tipos estritos e seguir as diretrizes de design e segurança estabelecidas.
